# 🛠️ Valley Express Prompt Builder (RICE & SCOPE)

## Overview
The Prompt Builder transforms casual instructions into high-fidelity, agentic directives optimized for the Antigravity architecture. It utilizes the twin frameworks of **RICE** (Core Identity) and **SCOPE** (Tactical Execution).

---

## 🏗️ The RICE Framework (Core Identity)
*Use this to define the "Soul" of the task.*

- **R - ROLE**: 
  - *Options*: [Sovereign Logistics Strategist, Aletheia Auditor, Neural Dispatcher, Social Media Commander]
  - *Prompt*: "You are the [Role]... [Describe expertise like Phoenix corridor mastery or SAM.gov protocols]."
- **I - INPUT**: 
  - *Prompt*: "The user wants to [Specific Action]... [Add details like lead count, target date, or platform]."
- **C - CONTEXT**: 
  - *Prompt*: "Valley Express is a [Brief Company Overview]. We operate on a 75/25 split. Current focus is [Medical/B2G/Industrial]."
- **E - EXPECTATIONS**: 
  - *Prompt*: "Deliver the output in [Format]. Tone must be [Corporate/Aggressive]. Quality standard: [e.g., No generic advice]."

---

## 🎯 The SCOPE Framework (Tactical Execution)
*Use this to define the "Steps" of the mission.*

- **S - SITUATION**: Describe the current technical state (e.g., "Web App is live," "Leads are ingested").
- **C - COMPLICATIONS**: List blockers (e.g., "Bot detection," "Missing SAM.gov API key").
- **O - OBJECTIVE**: State the high-level goal (e.g., "Convert 5 bids," "Launch social strike").
- **P - PLAN**: Step-by-step checklist (1. Research, 2. Audit, 3. Push, 4. Verify).
- **E - EVALUATION**: Define success (e.g., "5 leads pushed to ledger").
