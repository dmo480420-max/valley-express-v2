# 🔱 Valley Express OS | Integrated Command System

## 1. Core Vision (from Brainstormer)
Valley Express is transitioning from a passive logistics firm to an **Autonomous Logistics Sniper**. 
- **Role**: Sovereign Logistics Strategist.
- **Identity**: ASU Sun Devil Pride (Maroon/Gold), White-Glove Excellence.
- **Target**: Phoenix Medical, Industrial, and B2G Procurement.

## 2. Technical Stack (v2.0 Migration)
- **Frontend**: Strike Console V2 (React/Next.js + Glassmorphism).
- **Backend**: Firebase Firestore (Real-time Lead Ledger).
- **Compute**: Firebase Functions (Autonomous Scraping Pulses).
- **Intelligence**: Google AI Studio (Gemini) for Lead Auditing (Aletheia Protocol).

## 3. Operational Workflow (Sniper Logic)
1. **Scout**: Automated scrapers (Metrobi, Craigslist, FB Groups) push raw leads to Firestore `leads_raw` collection.
2. **Audit**: `GeminiAuditor` triggers on new Firestore documents.
   - Assigns **Truth Score** (Aletheia Protocol).
   - Categorizes by Sector (Medical/B2G/Industrial).
3. **Dispatch**: Verified leads (Score > 80) appear in the **Strike Console**.
4. **Strike**: User triggers bid generation via **Stirling PDF** bridge.
5. **Sync**: Finalized contracts are synced to the **AppFlowy Sovereignty Hub**.
