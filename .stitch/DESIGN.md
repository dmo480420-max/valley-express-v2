# 🚛 Valley Express — Design System (Stealth Ops)

## 🎯 Global Aesthetic
- **Branding**: "Amber-Slate" Tactical Command Center
- **Atmosphere**: Professional, industrial, sovereign, high-intelligence.
- **Imagery**: Grayscale industrial landscapes with amber overlays. Tactical grids.

## 🎨 Color Palette
- **Deep Slate**: `#121212` (Background)
- **Amber Gold**: `#FFBF00` (Accents / CTAs)
- **Command Red**: `#FF3333` (Critical / HOT loads)
- **Glass Panel**: `rgba(255, 255, 255, 0.03)` with `backdrop-filter: blur(24px)`

## 🖋️ Typography
- **Headings**: *Outfit* (Geometric, Semi-expanded) - Heavy weights (800+)
- **Stats/Data**: *JetBrains Mono* (Technical/Terminal)
- **Body**: *Inter* (Clean, high-legibility)

## ⚙️ Design Tokens for Stitch Generation
- Use `border: 1px solid rgba(255, 255, 255, 0.08)` for all panels.
- Buttons must have `box-shadow: 0 0 20px rgba(255, 191, 0, 0.3)`.
- Apply a fixed grid background: `linear-gradient(rgba(255,255,255,0.05) 1px, transparent 1px)`.
- Use **Industrial Icons** (boxes, trucks, radars, shield).
