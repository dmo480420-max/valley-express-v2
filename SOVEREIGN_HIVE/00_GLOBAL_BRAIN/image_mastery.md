## 🎨 #G_AND_CO_DESIGN_STANDARDS

### High-Impact Minimalism (HIM)
- Every pixel has purpose — no visual clutter, no filler elements
- Zero "AI slop": no plastic skin, no floating hands, no misaligned reflections
- Composition follows rule of thirds; negative space is intentional, not accidental
- If it doesn't serve the story, it doesn't exist in the frame

- **Biophilic lighting**: Directional natural light — window light, golden hour, canopy dapple

- **No harsh flash**: Avoid clinical lighting unless medical/lab context is specified
- **Shadows**: Soft, directional — never flat or absent

## 🌍 #GEO_MAPPING_PROTOCOL (ZERO-FRICTION)

Before generating any location-specific image, verify the physical world. Do not guess.

### Verification Checklist
- [ ] **Landmarks**: Confirm specific identifiable structures within frame distance
- [ ] **Layout**: Verify street orientation, building setbacks, lot size aesthetics
- [ ] **Climate zone**: Match vegetation, sky tone, and atmospheric haze to season + region
- [ ] **Signage materials**: Match to local code (e.g., desert Southwest = sandstone/adobe; Pacific NW = cedar/Douglas fir)
- [ ] **Native trees/plants**: Use regionally accurate species (e.g., Saguaro for Phoenix, Live Oak for Austin, Magnolia for Atlanta)
- [ ] **Light angle**: Adjust sun position to latitude + time of day specified

### Geo Tags (Apply to Prompt)
```
GEO: [City, State] | SEASON: [Spring/Summer/Fall/Winter] | TIME: [Dawn/Morning/Midday/Golden Hour/Dusk/Night]
```

### Example
```
GEO: Phoenix, AZ | SEASON: Fall | TIME: Golden Hour
→ Sky: warm amber gradient, low sun angle from the west
→ Vegetation: Palo Verde trees (dormant), Saguaro cactus, Brittlebush
→ Materials: sand-colored stucco, terracotta tile, wrought iron accents
→ Light: long warm shadows, no harsh midday glare
```

## 🔒 #IDENTITY_LOCK_PROTOCOL

Used when a person must appear consistently across multiple images (owner, staff, brand ambassador).

### Setup
1. Assign a unique **Identity Label** at project kickoff
   - Format: `[Name] MST-[1–10]`
   - Example: `James MST-6` | `Aaliyah MST-4` | `Marco MST-8`
2. Define **Anchor Points** — stored in project config, referenced in every prompt:

| Anchor | Specification |
|--------|--------------|
| Jawline | Square / Oval / Defined / Soft — pick one, lock it |
| Eye spacing | Narrow / Medium / Wide |
| Nose structure | Broad / Narrow / Straight / Rounded tip |
| Skin tone | MST scale 1–10 (Monk Skin Tone standard) |
| Heritage | Explicit multi-racial or monoracial spec |
| Hair | Style, texture, length — locked unless explicitly changed |
| Build | Lean / Athletic / Broad — consistent across frames |

### Consistency Rules
- **Body posture** must match brand role across all frames (authoritative for owner, collaborative for staff)
- **Attire continuity**: same outfit within a session unless `CHANGE: outfit` command is issued
- **Realism enhancers**: include at least 2 environmental grounding details per frame (see below)
- Identity label must appear in every prompt that features this person

### Realism Enhancers (Mandatory per Frame)
Prevents the "AI plastic" look. Include ≥2 per image:
- Ceiling fan, HVAC vent, track lighting rig
- Desktop items (coffee cup, notebook, phone face-down)
- Fabric texture on clothing (wrinkles, weave visible)
- Ambient background sounds implied visually (open window, soft light shaft)
- Natural skin detail: pores, slight brow asymmetry, lip texture

## ✏️ #PRECISION_EDITING_FORMULA

All image edit instructions must use exactly one of five approved verbs. No exceptions.

| Verb | Usage | Example |
|------|-------|---------|
| **ADD** | Insert a new element that was not present | `ADD: Diamond Blue pocket square` |
| **CHANGE** | Modify an existing element's property | `CHANGE: background to charcoal walnut wall` |
| **MAKE** | Transform the overall feel or state | `MAKE: lighting warmer, golden hour tone` |
| **REMOVE** | Eliminate an element entirely | `REMOVE: logo on the wall behind subject` |
| **REPLACE** | Swap one element for another | `REPLACE: white shirt with tobacco linen blazer` |

### Rules
- One verb per instruction line — stack multiple lines for compound edits
- Never use: "adjust," "tweak," "fix," "update," "improve" — too vague for image engines
- Always specify the element AND its target state

### Example Edit Block
```
CHANGE: jacket color to cognac leather
ADD: vintage wristwatch, brushed gold case, brown leather strap
REMOVE: overhead harsh fluorescent light
MAKE: overall tone warmer, 35mm film grain overlay
REPLACE: plain wall with charcoal wood panel texture
```

## 📷 #CAMERA_CONTROL_SYSTEM

### Focal Length Guide
| Focal Length | Use Case | Character |
|---|---|---|
| **35mm** | Lifestyle, environment, candid | Wide context, slight perspective distortion, feels "in the room" |
| **50mm** | Balanced — product + people | Natural perspective, closest to human eye, editorial |
| **85mm** | Portrait compression | Flattering compression, background separation, luxury feel |
| **135mm** | Tight headshot / detail | Extreme compression, bokeh-heavy, high-fashion editorial |

### Depth of Field Rules
- **Subject**: Always sharp — zero motion blur unless intentionally specified
- **Background**: Contextually soft — level of blur scales with focal length
  - 35mm → subtle background softness
  - 85mm+ → creamy bokeh, background abstracted to color fields
- **Foreground elements**: Slightly soft when used for depth framing (e.g., out-of-focus plant leaf in corner)

### Shot Type Reference
```
SHOT: [Focal Length] | ANGLE: [Eye Level / Low / High / Dutch] | FRAME: [Full Body / Mid / Bust / Headshot]
```

### Example Camera Block
```
SHOT: 85mm | ANGLE: Eye Level | FRAME: Bust
→ Shallow depth of field, subject razor-sharp
→ Background: charcoal walnut wall, softly abstracted
→ Foreground: slight biophilic frame (out-of-focus leaf, lower left)
→ Lighting: directional window light from camera-left
→ Film grain: 35mm texture overlay
```
